This folder contains three java projects.



1. PotholeMapper: The app for detecting the pothole.

2. Potholetest: The junit testing project for PotholeMapper.

3. DataGatherer: The project for collecting the data for analysis for development of algorithms for detection. 

If you are a developer and trying to install and run this code. Do read Install.txt before building the src code.
